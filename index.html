Object Finder using Android and Bluetooth

-Usually, people get frustrated when they lose something such as wallets, keys, pen drives, laptops, etc. The main objective of this project is to help people who lose their items frequently and also avoid thievery of the items.

-To achieve this objective, we are developing an application on the Android OS for communicating with a Bluetooth tag that can be attached to the concerned items.

What is Bluetooth Low Energy?

-Hit the market in 2011 as Bluetooth 4.0.

-The key difference is in Bluetooth 4.0's low power consumption.

-With Bluetooth LE's power consumption, applications can run on a small battery for four to five years.

-This is vital for applications that only need to exchange small amounts of data periodically.

-Just like Bluetooth, BLE operates in the 2.4 GHz ISM band. Unlike classic Bluetooth, however, BLE remains in sleep mode constantly except for when a connection is initiated. The actual connection times are only a few ms, unlike Bluetooth which would take approximately 100ms.

BLE can communicate over 40 channels from 2.4000 GHz to 2.4835 GHz. 37 of these channels are used for connection data and the last three channels (37, 38, and 39) are used as advertising channels to set up connections and send broadcast data.

BLE uses a technique called frequency hopping spread spectrum, in which the radio hops between channels on each connection event. The value of the hop is communicated when the connection is established, so it is different for every new established connection. This technique minimizes the effect of any radio interference.

-Program HM 10 module and add device driver using TTL to USB converter.

-The BLE module is put in the slave mode and the phone's Bluetooth module will be the master.

-Power Supply Unit (PSU) powers the bluetooth module.

-The module will be paired before attaching to any item.

-If the item is lost, we can start the app and get to know how near or far we are from the item

-The BLE module sends the RSSI (Receiver Signal Strength Information).

-The proximity of the lost item will correspond to the BLE module. Distance can be approximated using RSSI.

Feasibility

-Smaller BLE modules like the BLE-112 can be used but presently they are not available for purchase.

-The estimated range is 80m, but this range is applicable only if there are no obstructions to the signal like walls, building, etc.

-Legal feasibility: It is legal to use BLE modules anywhere.

-Another important point is if the item gets lost then someone else with the same application may be able to get hold of your item.
